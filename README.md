# SAQAYA E-Commerce

A simple e-commerce app developed in Vue 3 with Typescript, Pinia and Vue Router

## Project Setup

### Go to project directory

```sh
cd saqaya-ecommerce
```

### Install dependencies

```sh
npm install
```

### Run the server for development

```sh
npm run dev
```

## Project structure

```sh
saqaya-ecommerce
├───public
└───src
    ├───assets
    ├───components
    │   ├───cart
    │   └───products
    ├───lib
    ├───router
    ├───store
    ├───types
    ├───views
    └───__tests__
```
